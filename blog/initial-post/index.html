업무/스터디를 진행하면서 어렵거나 중요하다고 생각되는 것들을 기록하기 위한 용도로 github 페이지를 새롭게 만들었다.

요즘 Vue.js 스터디를 하고 있어서 Vue.js 기반으로 만들어진 website building framework중 하나인 Gridsome을 골랐다. (다른 예로 VuePress와 Nuxt.js도 있다.) Javascript와 Vue.js를 잘 몰라도 html/css만 조금 알고 있으면 누구나 나처럼 블로그를 만들수 있고, 원하는 테마도 설정할 수 있다. 내가 참고한 starter theme는 Kendall Strautman의 Brevifolia이다.

앞으로 열심히 기록해야지 ^___^
